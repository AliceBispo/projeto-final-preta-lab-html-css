  • Code Up

    Code Up

    Ajudando mulheres a se encontrar e transformar sua aptidão em realidade. O projeto Code Up nasceu de uma iniciativa de mulheres, discentes da UFRB, com intuito de criar um grupo de mulheres aprendendo e programando juntas para difundir a tecnologia e aplicar conhecimento prático. O grupo se estende a um público cujo interesse seja o desenvolvimento de tecnologia, não importando a área de atuação. Nosso objetivo é uma maior difusão da tecnologia e o estudo da linguagem de programação entre as mulheres da UFRB e do Recôncavo Baiano. Aprendento a programar do zero.
